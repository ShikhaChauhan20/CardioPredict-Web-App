# 🫀 Heart_Disease_Prediction_ML

A full-stack Machine Learning application that uses clinical patient data to predict the risk of heart disease. The project features a predictive classification model deployed with a clean, interactive user interface.

▶️ **[Witness it Live Here!](https://heart-disease-predictor-gs3c.onrender.com)**

---

## 🛠️ Features & Tech Stack

* **Machine Learning Pipeline:** Built using classification models (Random Forest / Logistic Regression) optimized in Python.
* **Interactive UI:** A responsive, modern web form using standard HTML, CSS, and the Nunito web font.
* **Flask Backend:** A lightweight Python web server connecting frontend form data to the ML prediction script in real time.
* **Cloud Deployment:** Hosted publicly on Render with live server handling.
